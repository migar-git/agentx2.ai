# SEO Strategy — Topic Clusters & Programmatic SEO

> **Breadcrumb:** [Home](../../README.md) › [Docs Index](../INDEX.md) › [Website](WEBSITE_ARCHITECTURE.md) › **SEO Strategy**
> **Status:** `Active` · **Owner:** `content-swarm` · **Last verified:** `2026-06-12`

## 1. Purpose

How AgentX2.ai earns organic discovery: topic clusters, authority pages, and a programmatic SEO
factory — all grounded in [Google Search Essentials](https://developers.google.com/search/docs/essentials)
and structured with [schema.org](https://schema.org/).

## 2. Topic clusters (pillar → spokes)

| Pillar | Spoke themes |
|--------|--------------|
| Agentic AI | agent design, multi-agent, digital workers, automation |
| AI for Finance | CFO copilots, FP&A automation, forecasting |
| AI Governance | responsible AI, model/agent governance, compliance |
| AI Automation | workflow automation, RAG, knowledge work |
| Industries | finance, healthcare, manufacturing, tech, prof. services |

Each pillar is an authority page; spokes interlink to the pillar and to each other (≤3 clicks),
reinforcing the [website IA](WEBSITE_ARCHITECTURE.md).

## 3. On-page SEO standard

- Unique title + meta description; canonical URLs.
- [schema.org](https://schema.org/) structured data (`Organization`, `Service`, `FAQPage`,
  `BreadcrumbList`, `Article`).
- Semantic headings, descriptive alt text, internal links.
- `sitemap.xml` ([protocol](https://www.sitemaps.org/protocol.html)) + `robots.txt`, auto-generated.
- Performance is an SEO input — meet [Core Web Vitals](PERFORMANCE.md).

## 4. Programmatic SEO

Templated, **grounded** pages for use-case × industry × persona combinations, generated by the
[Content Factory](CONTENT_FACTORY.md). Each page must add real value (no thin/duplicate content) and
passes the [Quality Gates](../04-quality/QUALITY_GATES.md) and a grounding check before publish.

## 5. Measurement

Rankings, impressions, CTR, and organic conversions tracked in
[Analytics](../05-observability/ANALYTICS.md); SEO regressions are caught by the
[Regression Policy](../04-quality/REGRESSION_POLICY.md).
